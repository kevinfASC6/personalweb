USCF Live Ratings Web Scraper

A tool that allows users to easily search up to 30 players given their USCF IDs and returns the information in under 60 seconds. I built this in response to the USCF websites slow loading time and which took up a lot of time when I was trying to check the ratings of my students or when I had to use live ratings for a chess tournament. Future plans are to make it more scalable and allow users to search for even more players, while also presenting more information. Version 1 is ready and can be accessed. Currently working on Version 2.
